# Baseball-Game
Java 숫자야구게임


## 🖥️ 프로젝트 소개
Java 언어에 익숙해지기 위한 숫자야구게임 만들기입니다.
<br>

## 🕰️ 개발 기간
* 23.06.13일 - 22.06.15일

### 🧑‍🤝‍🧑 맴버구성
- 팀장  : 강영준 - 컴퓨터 랜덤숫자 생성 및 중복확인
- 팀원1 : 강영준 - 유저 입력 생성 및 중복확인
- 팀원2 : 강영준 - 컴퓨터 숫자와 유저 숫자 비교 및 결과 출력

### ⚙️ 개발 환경
- `Java 8`
- `JDK 1.8.0`
- **IDE** : STS 3.9
- **Framework** : Springboot(2.x)
- **Database** : Oracle DB(11xe)
- **ORM** : Mybatis
